📂 Tab 1

File Explorer

A built-in file browser designed for image assets. Drag & drop a folder to open it, then browse, sort, and preview without leaving the app.

  • ✓ Drag & drop folders or click to browse
  • ✓ Three view modes: Details, Medium thumbnails, Large thumbnails
  • ✓ Sort by name, file size (↑↓), or resolution (↑↓)
  • ✓ Live image preview panel with auto-scaling
  • ✓ Shows file size, type, and resolution at a glance
  • ✓ Navigate with Back / Up buttons and breadcrumb path
  • ✓ Supports PNG, JPG, JPEG, and WebP
✂️ Tab 2

Smart Image Trimmer

Automatically crop away transparent borders and optionally remove solid-color backgrounds. Perfect for cleaning up exported sprites and icons.

  • ✓ Alpha Threshold — transparency detection aggressiveness (0–254)
  • ✓ Color Trim — remove pixels matching a target color
  • ✓ Fuzz Tolerance — how close a color must be to get trimmed (0–255)
  • ✓ Padding — add pixels back so edges aren't clipped
  • ✓ Batch process folders recursively
  • ✓ Overwrite originals or save to trimmed/ subfolder
  • ✓ Real-time log with before/after dimensions
📐 Tab 3

Image Resizer

Batch-resize all your images to a consistent size while preserving aspect ratio. Seven modes cover every use case.

  • ✓ Fit (no upscale) — shrink to fit a box, never enlarge
  • ✓ Fit + Upscale — also scales up small images
  • ✓ Exact Canvas — center on a transparent W×H canvas
  • ✓ Long Edge / Short Edge — scale based on one dimension
  • ✓ Width Only / Height Only — lock one axis
  • ✓ Presets: 128², 256², 512², 1024², 2048², 1920×1080, 1280×720
  • ✓ Output as PNG, JPEG, WebP or keep original format
  • ✓ Filters: LANCZOS, BICUBIC, BILINEAR, NEAREST (pixel art)
🧠 Tab 4

Background Remover

Remove backgrounds from any image using AI. Powered by rembg with 7 ONNX models.

  • ✓ 7 AI models — general, anime, human, clothing, and more
  • ✓ Live status indicators for installed models
  • ✓ Alpha Matting for cleaner edges (slower but better)
  • ✓ Auto-trim after removal using smart trim
  • ✓ Batch process folders with recursive support
  • ✓ Output organized into model-named subfolders
  • ✓ Progress bar and detailed per-file log
⚙️ Tab 5

Install Manager

Manage rembg installation and AI model downloads from within the app. Each action opens a real terminal window.

  • ✓ One-click rembg installation via pip
  • ✓ Per-model install buttons that open CMD/Terminal
  • ✓ Real-time status: ✅ installed, ❌ missing, ⏳ checking
  • ✓ Refresh button to re-scan all model statuses
  • ✓ Shows Python path, model folder, platform info
  • ✓ Quick button to open model folder in file manager
  • ✓ Cross-platform: Windows, macOS, Linux
🧠 AI Model Downloads

Download Pre-trained Models

Download the ONNX model files directly and place them in your model folder. These are the four primary models — additional models can be installed from the app.

📁 Install Location: After downloading, place the .onnx file in:
Windows: C:\Users\YourName\.u2net\
macOS/Linux: ~/.u2net/

Or use the ⚙️ Install tab inside the app to download automatically.
🚀

High Precision

isnet-general-use

Best overall quality for general-purpose background removal. Works great with complex edges and fine details.

📥 Getting Started

Installation Guide

Get up and running in under 5 minutes.

1

Install Python 3.8+

Download from python.org. Check "Add Python to PATH".

python --version
2

Install Required Packages

Open a terminal and install dependencies:

pip install Pillow numpy

Optional — for drag & drop support:

pip install tkinterdnd2
3

Download the App

Clone or download from GitHub:

git clone https://github.com/AmirhosseinAlimirzaei/2D-Asset-Toolkit.git cd 2d-asset-toolkit
4

Run the App

python toolkit.py
5

Install rembg & AI Models (Optional)

For Background Remover, go to the ⚙️ Install tab inside the app, or manually:

pip install "rembg[cpu,cli]"

Then download models from the Models section above.
